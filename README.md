# js2go
Implement golang cosmos-sdk method using js with help of [gopherjs](github.com/gopherjs/gopherjs) library

#Pre-requirents

**golang** should be installed

#install
```$xslt
go get -u github.com/baymax19/js2go
```

#Transactions
- createKey
- sendCoins


build the main.go using
```$xslt
gopherjs main.go 
```
